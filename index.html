Habilidade como Analista de Sistema

Me dá um problema e lhe dou uma solução, se vier com a solução lhe dou uma ideia de como melhorar ela como uma equipe. O meu papel é procurar e melhorar a ideia por meios de metodologia cientifica e casos de uso, para ter uma boa usabilidade da aplicação, vou usar métodos de como chegar a 99,9% da solução.

Com os requisitos certos chegaremos a uma conclusão e objetiva para resolver o problema de forma rápida e eficiente, por dentro das tecnologias atuais vou flexionar o mais rápido passível para entregar o projeto.

Como os projetos são feitos ?

Os projetos são passados por uma sequência de requisitos por pesquisar feita na base das informações que o cliente vai dar. Depois que colher as informações vamos separar o que é importante para o funcionamento do projeto, com exceções de casos de uso para fazer o teste de como o projeto vai se comportar durante o uso com outras pessoas em geral, depois que chegar na metade da análise vamos procurar a melhor tecnologia para trabalhar e criar uma nova performance para que ele seja prático e fácil de usar e ter a principal função que é a confiabilidade.

Para finalizar vamos colocar na fase experimental antes de subir para produção isso vai evitar que o projeto represente bugs, enfim depois que terminar a trajetória vamos subir o projeto em produção e fazer a manutenção do site assim que o cliente solicitar.
